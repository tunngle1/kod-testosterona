# Код тестостерона

Лендинг бесплатного онлайн-вебинара Александра Евсеева «Код тестостерона».

## Запуск

Откройте `index.html` в браузере или разместите файлы на любом статическом хостинге.

## Структура

- `index.html` — страница лендинга
- `css/style.css` — стили
- `js/main.js` — форма регистрации и меню
- `images/` — логотип и фото
